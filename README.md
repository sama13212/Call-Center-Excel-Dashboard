# Call Center Performance Dashboard (Excel)

This project analyzes over **5,000 call center records** to evaluate agent performance, customer satisfaction, call resolution, and operational efficiency.

The dashboard was built using **Microsoft Excel** with **Power Query** for data cleaning, **Data Model** for relationship management, and **DAX Measures** for KPI calculations. Interactive slicers allow users to filter the dashboard by agent, topic, and other dimensions.

### Key Features

* Cleaned and transformed raw data using Power Query.
* Built a Data Model to optimize analysis.
* Created DAX Measures for dynamic KPIs.
* Designed interactive dashboards with Pivot Tables and Slicers.
* Analyzed call trends, agent performance, and customer satisfaction.

### Key Insights

* Jim handled the highest number of calls (666).
* Dan achieved the lowest average handling time (7.5 minutes).
* Streaming generated the highest call volume (1,022 calls).
* The overall resolution rate reached 81%, leaving 946 unresolved calls.
* Average customer satisfaction was 3.4/5.
* Monday recorded the highest call volume during the analyzed period.

### Recommendations

* Improve resolution rates by investigating unresolved cases.
* Focus training efforts on high-volume call topics.
* Optimize staffing during peak periods, especially on Mondays.
